mysql에서 같은 테이블의 필드값을 다른 필드로 복사 할 경우가 있습니다.
해당 경우는 아래와 같은 쿼리로 값을 복사 할 수가 있습니다.

* table name : table_a
* new_field : 데이터를 복사할 새(다른)필드
* old_field : 기존 필드

update table_a A INNER JOIN table_a B
on A.ID = B.ID
set A.new_filed = A.old_field
where A.ID = B.ID;
