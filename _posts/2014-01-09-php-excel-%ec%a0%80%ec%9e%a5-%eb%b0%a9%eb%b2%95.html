php에서 excel로 export 하는 방법

아래 코드를 추가하고

 

<?php
$file_name = date("Y-m-d") . "-리스트.xls";
		    header("Pragma:");
		    header("Cache-Control:");
		    header("Content-Type: application/vnd.ms-excel");
		    header("Content-Disposition: attachment; filename=$file_name");
		    header("Content-Description: PHP5 Generated Data");
?>

 

필요하다면 db에서 data를 불러와서 html table에 뿌려주면 자동으로 엑셀파일로 다운로드 됩니다.
아래 style 을 추가한 이유는 숫자가 길어질 경우 데이터 원본대로 뿌려주기 위한 style 입니다.
필요하신 행에 추가해주시면 됩니다.

 

<html> 
<style> .banknumber{ mso-number-format:\@; } </style>
<body>
	<table cellspacing=0 cellpadding=0 border=1>
          <tr align='center'>
		<td>자료1</td>
		<td class="banknumber">자료2</td>
          </tr>
	</table>
</body> 
</html>
