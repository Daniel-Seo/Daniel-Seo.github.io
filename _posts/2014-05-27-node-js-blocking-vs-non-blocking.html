  • Blocking Code

    Read file from Filesystem, set equal to "contents"
    Print contents
    Do something else

    var contents = fs.readFileSync('/etc/hosts'); // 해당 파일을 읽는다
console.log(contents); // 해당 파일을 다 읽으면 contents를 출력한다
console.log('Doing something else'); // contents를 다 출력하고 로그를 출력한다

     

  • Non-Blocking Code

    Read file from Filesystem
      whenever you're complete, print the contents ("callback")
    Do something else

    var callback = function(err, contents) {
    console.log(contents);
}
fs.readFile('/etc/hosts', callback); // 해당 파일을 다 읽으면 callback 한다
console.log('Doing somethings else'); // 만약 위에 해당 파일의 크기가 커서 다 읽지 않더라도 로그가 출력된다.
